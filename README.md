# RKN Block Checker

Инструмент для анализа доступности хостинга и проверки методов блокировок (РКН). Позволяет владельцам ресурсов и системным администраторам диагностировать причины недоступности сайта и определять используемые механизмы фильтрации трафика.

## Возможности

### Базовые проверки связи
*   **Security Audit:** Проверка заголовков безопасности (HSTS, CSP).
*   **SSL Deep Checker:** Глубокий анализ SSL-сертификата (срок действия, эмитент, протоколы).
*   **Port Scanner:** Проверка открытых стандартных портов (21, 22, 80, 443, 3306, 3389).
*   **Tracert Test:** Трассировка маршрута до целевого узла.
*   **Ping (8KB/16KB):** Проверка прохождения пакетов увеличенного размера (L3/L4 блокировки).

### Внешние сервисы
*   Интеграция с **BGP.he.net**, **Censys.io** и **BGP.tools** для анализа автономных систем и маршрутизации.

### Анализ методов блокировки (РКН)
*   **TCP RST Check:** Детектирование принудительных разрывов соединений (Connection Reset).
*   **QUIC Check:** Проверка доступности протокола HTTP/3 (QUIC).
*   **Block Transfer (8KB/16KB):** Тестирование блокировок на основе анализа содержимого пакетов (DPI) при передаче данных определенного размера.

## Требования

*   [Go](https://go.dev/dl/) 1.21 или выше.
*   Права администратора/root (для некоторых типов пингов и трассировки).

## Установка и запуск

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/FlexEbat/rkn-block-checker.git
   cd rkn-block-checker
   ```

2. Установите зависимости:
   ```bash
   go mod tidy
   ```

3. Запустите приложение:
   ```bash
   go run main.go
   ```
